Guidelines and specifications

The source code and specifications for Grid Engine are posted under the The Sun Grid Engine Software License and are free for you to use according to the license. Sun Grid Engine Users Manuals are also available in PDF format.

Get involved

As with any open source project, our goal is to get the community involved to both use Grid Engine and to contribute. There are many ways of contributing to an open source software project. Here are a couple of ways that you can immediately get involved:
  • Join a mailing list - Join the mailing lists to post general feedback, feature requests, and requests for help. Also, if you see a question on the list that you can answer, pleased do! See the mailing lists page for details on how to subscribe.

  • Join the project as Observer - Join the Grid Engine project as observer, you can submit bug reports and feature requests using Issuezilla, you have read only CVS access. See the project membership request page for applying as observer.
    • We welcome your feedback, comments, and your contributions. For more information, view Grid Engine Frequently Asked Questions.
