Globus Integration

A Globus integration package for Sun Grid Engine has been developed by Marko Krznaric, Department of Computing, Imperial College London, UK. The information can be found at his web page.


An alternative integration package for Globus Toolkit and Grid Engine has also been developed by GridwiseTech team. See more information on GridwiseTech web page.
