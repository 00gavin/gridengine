GRID ENGINE DOCUMENTS


Two categories of documentation are available for Grid Engine, user/administrator level manuals for the compiled software and developer oriented technical documentation. Some of those files are available in PDF format. You will need Adobe®Acrobat®Reader®to display them.Download Acrobat Reader®.

User and Administrator Manuals

The Grid Engine project has no targetted effort setup yet to create documentation tailored to the need of Grid Engine. However, a draft version of a user and administrator manual for a yet unreleased Sun product exists which describes the functionality of the initial open source release of Grid Engine reasonably well. It is available here because it certainly will be useful to those who are interested in using Grid Engine in their environment.
The manual is available as PDF only and is 12.5 MBytes big. To download it click here. It consists of

  • a Quick Start Guide including installation instructions,
  • an Administration and Installation Manual,
  • an User Guide, and
  • a Reference Manual.
In addition, "nroff" on-line manual pages can be checked out from /gridengine/doc/man. Click here for instructions on checking out from CVS. They contain the same information as the Reference Manual. This includes a description of all client/server commands, of all file formats and of the internally used programming interfaces.
For administrators and users we also recommend to check our HOWTOs, specifically in case of problems or questions concerning how Grid Engine might be best applied to a specific scenario.

Technical Documentation

Those who are interested in understanding the Grid Engine source code or contributing to it are refered to the module documentation overview page. It contains links to information pages on all major source code modules and it provides an overview on their interrelationship.

Other sources of technical information are:

  • the "nroff" manual pages in HTML,
  • some source module specific documentation which can be found in the same directories as the module source code.

Developer Documentation

Additional documentation for developers is available:

  • a Grid Engine build page describing how to create Grid Engine binaries from the sources and a distribution,
  • a README.aimk describing the aimk scirpt in detail refered in the Grid Engine build page,
  • a scripts/README.distinst describing the distinst script in detail refered in the Grid Engine build page,
  • a scripts/README.mk_dist describing the mk_dist script in detail refered in the Grid Engine build page,
  • a sge_depend describing the tool used for generating make dependencies. It is refered in the Grid Engine build page,
  • a dist/README.arch describing the binary architectures of Grid Engine,
  • a scipts page describing other scripts which are needed during the build and installation process,
  • a testsuite page describing the testsuite used for quality assurance for the Grid Engine,
  • a coding standards,
  • a ADOC page describing Autodoc Headers used for source code documentation in more and more parts of the source code,
