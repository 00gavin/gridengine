GRID ENGINE DOCUMENTS


Two categories of documentation are available for Grid Engine, user/administrator level manuals for the compiled software and developer oriented technical documentation. Some of those files are available in PDF format. You will need Adobe®Acrobat®Reader®to display them.Download Acrobat Reader®.

User and Administrator Manuals

The Grid Engine manual set is available in PDF. It consists of

  • a Quick Start Guide including installation instructions,
  • an Administration and Installation Manual,
  • an User Guide, and
  • a Reference Manual.
In addition, "nroff" on-line manual pages can be checked out from /gridengine/doc/man. Click here for instructions on checking out from CVS. They contain the same information as the Reference Manual. This includes a description of all client/server commands, of all file formats and of the internally used programming interfaces.

The Adobe® FrameMaker® "source" files to the manual set are also available for checkout from /gridengine/doc. Please refer to the documentation guide (PDF) for more information on the manual sources.

Technical Documentation

Those who are interested in understanding the Grid Engine source code or contributing to it are refered to the module documentation overview page. It contains links to information pages on all major source code modules and it provides an overview on their interrelationship.

Other sources of technical information are:

  • the "nroff" manual pages described above,
  • some source module specific documentation which can be found in the same directories as the module source code
  • a build page describing how to create binaries from the sources.
