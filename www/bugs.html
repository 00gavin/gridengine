Reporting Bugs
The Grid Engine project is currently using IssueZilla, to track bugs. IssueZilla is an open source bug tracking system developed and used by Mozilla. You use IssueZilla to browse or modify existing bugs, or enter new bug reports. 

For information on how to submit bugs for Grid Engine, click here.

To submit an issue for a project, do the following:

  1. View the list of projects from the View Projects page or the My Projects page.
  2. Select a project from the list, and click on Issue Tracking from the toolbar.
  3. Select Defect under "Enter an Issue", and complete the form.
For more information, refer to IssueZilla issue tracking.
