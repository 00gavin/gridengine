Announcing http://gridengine.info

Several members of the Grid Engine user and developer community are contributing to a new website called "gridengine.info". The site was set up to address the growing amount of Grid Engine related news, usage, administration and "best practices" information often found deep within mailing list archives and in scattered locations across the internet.

One particular focus is going to be highlighting and deep linking into the archives of Grid Engine user and developer mailing list that do not currently get indexed by major internet search engines. A large body of practical Grid Engine administration, usage and configuration information is available via the mailing lists and the new portal site will be used to highlight, excerpt and link to interesting posts and discussion threads.

The site:

http://gridengine.info

A syndicated RSS feed is available at:

http://gridengine.info/xml/rss20/feed.xml

The new website is a volunteer effort set up with no official ties to Sun N1 Grid Engine or the primary Grid Engine http://gridengine.sunsource.net hosting site. Comments and feedback regarding http://gridengine.info should be addressed to Chris Dagdigian, <dag@sonsorol.org>.
