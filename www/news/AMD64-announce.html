Announcing Grid Engine availability for Linux on AMD Opteron

Regensburg, Germany -- February 10, 2004



Grid Engine is now available as a native 64-bit port for AMD Opteron systems running a Linux kernel 2.4.x.

Courtesy binaries for this architecture will be available by 02/12/04 on the project download page.

Users who want to access the corresponding source code need to checkout the CVS tag

V53p5AMD64_TAG
in the branch of the CVS repository
V53_beta2_BRANCH
A snapshot of the Grid Engine 5.3p5AMD64 sources is available as a source tarball at the  File Exchange Tool as file sge-V53p5AMD64_TAG-src.tar.gz.
