Announcing Grid Engine availability for Mac OS/X

Regensburg, Germany -- November 21, 2003


Grid Engine is now available for the Apple Macintosh operating system Mac OS/X ("Darwin"). Users from the Grid Engine open source community (among them Derek Homeier, Ron Chen, William van Etten, Brooks Davis, Greg Wuller and others)  began to port Grid Engine more than two years ago to Mac OS/X. Since then many sites have run Grid Engine successfully in their environments.

The generous donation of a development machine from Apple Computer now makes it possible for Sun Microsystems to add courtesy binary support  for this platform. Binaries for Mac OS/X are now available from the project download page. Users who want to access the corresponding source code need to checkout the CVS tag

V53p5_TAG
in the branch of the CVS repository
V53_beta2_BRANCH
A snapshot of the Grid Engine 5.3p5 sources is available as a source tarball at the  File Exchange Tool as file sge-V53p5_TAG-src.tar.gz.
