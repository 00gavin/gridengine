Grid Engine 6.2u1 and Hedeby (SDM) 1.0u2 Courtesy Binaries now Available for Download

Regensburg, Germany -- Dezember 18, 2008

Grid Engine 6.2 Update 1 is now ready and courtesy binaries are available for download. SGE 6.2u1 is a maintenance release and fixes bugs of the software, installation procedure and man pages.

The courtesy binaries are available at:

http://gridengine.sunsource.net/downloads/latest.html

A list of fixed problems since SGE 6.2 beta2 is available at

http://gridengine.sunsource.net/gridengine/62patches.txt

The patch installation notes are available at

http://gridengine.sunsource.net/gridengine/install62patch.txt

Please send support questions and feedback to the "users" mailing list of the Grid Engine open source project.

The courtesy binary distribution is now hosted on Sun Microsystems servers on Sun Download Center (SDLC).

The corresponding source code tag in the CVS repository has the name
         V62u1_TAG (Grid Engine and ARCo) and V10_U2_TAG (Service Domain Manager)

A snapshot of the sources is available at the Documents & Files page
        arco-V62u1_TAG-src.tar.gz
        ge-V62u1_TAG-src.tar.gz
        hedeby-V10_U2_TAG-src.tar.gz

The documentation and release notes are available at
        http://wikis.sun.com/display/GridEngine
