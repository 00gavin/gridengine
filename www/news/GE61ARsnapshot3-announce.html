Announcing Grid Engine 6.1 Advance Reservation Snapshot 3

Regensburg, Germany -- November 2, 2007

Grid Engine 6.1 Advance Reservation Snapshot 3 is now ready and courtesy binaries are available for download. This release is aimed for testing purposes only and not for productive clusters.

This snapshot is based on the Grid Engine 6.1 Update 2 release enhanced with the Advance Reservation (AR) feature. AR will be included in the next major update release of Grid Engine. The functionality allows users or administrators to reserve particular resources for future use. These reserved resources are only available for special jobs and the scheduler ensures the availability of the resource when the start time is reached. The design of the Grid Engine Advance Reservation can be found here:

http://gridengine.sunsource.net/source/browse/*checkout*/gridengine/doc/devel/rfe/AdvanceReservationSpecification.html

Note: This snapshot is not compatible with prior Grid Engine snapshots or versions. An upgrade procedure is currently not available.


The courtesy binaries are available at:

http://gridengine.sunsource.net/project/gridengine/downloads/61ar/download.html

Please test this snapshot and send support questions and feedback to the "users" mailing list of the Grid Engine open source project.
