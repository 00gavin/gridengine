Announcing Grid Engine 6.0 Update 7

Regensburg, Germany -- Dec 09, 2005

Grid Engine 6.0 Update 7 is now ready and courtesy binaries are available for download. The SGE 6.0u7 release includes performance improvents and fixes bugs of the software, installation procedure and man pages.

The major scalability improvements are:

  • qmaster GDI GET request handling (e.g. qstat) is now faster and consumes less memory.
  • async communication protocol between scheduler and master. This leads to a more efficient communication flow.
  • better commlib thread concurrency through introduction of RW-locks.
  • significantly enhanced DRMAA submission rate.
