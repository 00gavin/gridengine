Announcing Grid Engine 6.0 Update1

Regensburg, Germany -- August 17, 2004


Grid Engine 6.0 Update1 is now ready and courtesy binaries are available for download. The SGE 6.0u1 release fixes bugs of the software, installation procedure and man pages. The list of fixed problems is available at

http://gridengine.sunsource.net/project/gridengine/60patches.txt
The courtesy binaries are available at:
 http://gridengine.sunsource.net/project/gridengine/download60.html
Please send your support questions and feedback to the "users" mailing of the Grid Engine open source project.

The corresponding source code tag in the CVS repository has the name

V60_TAG
A snapshot of the sources is available at the File Exchange Tool :
