Announcing OpenDSP project

Web Service multi-user access to SGE over DRMAA for remote job submission and control

Poznan -- May 11 2006

The OpenDSP developers are happy to announce that an additional service exposing the Web Service multi-user access for job submission, control, and monitoring DRMAA routines in SGE 6.0u4 or later is available now as open source and 100% free.

In order to download OpenDSP sources and manual please visit the following webpage:

http://sourceforge.net/projects/opendsp

Please join our end users mailing list:

http://hedera.man.poznan.pl/mailman/listinfo/dsp-users

The OpenDSP issue tracker is also available:

http://hedera.man.poznan.pl/flyspray/

Release notes (OpenDSP v0.1 RC1):
  • Built-in support of DRMAA job submission, control, monitoring and error handling over DRMs
  • Authentication and encryption mechanisms based on SSL or GSI (Grid Security Infrastructure)
  • Successfully tested with external services based on Globus Toolkit 2.4.x, 3.2.x and 4.0
  • Authorization callouts and external access control list
  • External accounting and auditing database available
  • Stage-in/out operations supported over DIME/MIME attachments
  • JSDL 1.0 POSIX based job description
  • Successfully tested with the following resource/cluster management systems: SGE (SGEv6u4, SGE v6u5, SGEv6u7, SGEv6u8 recommended)
    • Linux x86, kernel 2.4, glibc >= 2.2
    • Linux AMD64 (Opteron), kernel 2.4, glibc >= 2.3
    • Solaris (Sparc) 9 64-bi
    • Solaris (x86) 9
    • Apple Mac OS/X
  • Java, .NET and C-based example clients and SDKs available
  • Distributed under the GPL license
