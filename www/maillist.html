GRID ENGINE MAILING LISTS

There are two types of mailing lists associated with the Grid Engine project community: "top-level" lists and "project" lists. The top-level lists are the primary means of communications on Grid Engine. The project lists are for CVS and Issuezilla system-generated messages.

Of course, which lists you subscribe to depends on what level of interest and involvement you have in the Grid Engine project.

Top Level Lists

announce@gridengine.sunsource.net
Description: A moderated mailing list for Grid Engine announcement and general messages
Subscribe: announce-subscribe@gridengine.sunsource.net
Unsubscribe: announce-unsubscribe@gridengine.sunsource.net

users@gridengine.sunsource.net
Description: A mailing list for discussing Grid Engine technology
Subscribe: users-subscribe@gridengine.sunsource.net
Unsubscribe: users-unsubscribe@gridengine.sunsource.net

Project Mailing Lists

To subscribe to a project specific mailing list, go to the mailing list page for that particular project. The different types of lists for each project are:

CVS
A mailing list for Concurrent Versions System (CVS) commit messages. This list receives an automated e-mail notification from CVS each time a developer commits code into the repository.

Issues
A mailing list for Issuezilla messages. This list receives an automated e-mail notification from Issuezilla each time a project member submits information assigned to the issues list.
