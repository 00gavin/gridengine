Grid Engine Mail Lists


There are two types of mailing lists associated with the Grid Engine project community: Mailing lists for discussion of user level issues and lists primarily interesting for developers. If you start installing and using Grid Engine the users mailing list will be your primary means of communication.

Of course, which lists you subscribe to depends on what level of interest and involvement you have in the Grid Engine project.

User Lists

announce@gridengine.sunsource.net (View Archive)
Description: A moderated mailing list for Grid Engine announcement and general messages
Subscribe: announce-subscribe@gridengine.sunsource.net
Unsubscribe: announce-unsubscribe@gridengine.sunsource.net

users@gridengine.sunsource.net (View Archive)
Description: A mailing list for discussing Grid Engine technology
Subscribe: users-subscribe@gridengine.sunsource.net
Unsubscribe: users-unsubscribe@gridengine.sunsource.net

gep@gridengine.sunsource.net (View Archive)

globus@gridengine.sunsource.net (View Archive)
