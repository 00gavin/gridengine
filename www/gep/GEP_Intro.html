Welcome to the Grid Engine Portal Home Page

The Grid Engine Portal project provides a java based capability for  enabling secure internet access to applications that run on an existing SGE grid.
Users authenticate to a portal interface from anywhere on the internet and can then securely:
  •    Execute applications via a transparent interface to Grid Engine
  •    Monitor the status of jobs running in Grid Engine
  •    Upload input files to the Portal Server with the click of a button
  •    Download output files to a local workstation with the click of a button
  •    View X-windows based applications using VNC
Administrators can also remotely access the portal and perform administrative functions like
  •    Registering applications for use with the GP in a matter of minutes
  •    Quickly building HTML interfaces to applications using templates
  •    Monitoring usage and statistics
In most cases this can be done without any modification of the existing application !
Furthermore, since applications can be executable scripts (and HTML interfaces can be easily built for scripts) this provides an efficient mechanism for handling complex job streams.

Since the variety of uses for such a capability are numerous, this package offers some basic features that many user communities should find beneficial. The source code was designed with the idea of making it available to the user community for further enhancements and capabilities. The Grid Engine Portal currently relies on GE as well as the SunONE Portal Server for its functionality although both are loosely coupled within the Grid Engine Portal source code.

The Grid Engine Portal (GEP) package can be downloaded as part of the Grid Engine Open Source Project.

Grid Engine Portal Components and Installation Overview

First of all the Grid Engine Portal installation requires that an existing SGE (version 5.3p2 or later) or SGEEE grid has been set up. For details on setting up SGE go to the web site http://wwws.sun.com/software/grid/. It is recommended that those unfamiliar with grid technology begin by taking the online Sun ONE Grid Technology Overview class. Administrators may also wish to take the online Sun ONE Grid Engine web-based Training.
Both of these courses are available at http://wwws.sun.com/software/gridware/training.html.

The next step is the installation of Sun ONE Portal Server (version 6 or later).
Note: there is also a Grid Engine Portal version available for version 3.0SP2 (or later) although it is recommended that users who do not have an existing Portal installation begin with version 6.
A trial/development package is included with the Solaris 9 disks. Alternatively you can download a development version here .
Sites that require remote visualization capabilities should download and install the Secure Remote Access package as part of this installation. Some installation guidelines are included in the complete Grid Engine Portal installation guide<link here>. This guide should be read before beginning to install the Portal Server.

Third the Grid Engine Portal software is available from the SGE open source site.
