Standard Versions


The following Grid Engine versions are currently considered Standard and are refered to by the Grid Engine Open Source License, in particular Exhibit B, and by the Grid Engine Compatibility Test. Standard Versions are used as a reference when determining compatibility of your modified version of Grid Engine. The Grid Engine Open Source License requires compatibility testing where you are providing your modified Grid Engine version to others.  You have to follow the conditions of the License and the Standards document referenced therein, and you have to pass the Grid Engine Compatibility Test against a Standard Version.
